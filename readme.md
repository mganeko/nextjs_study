# Study Next-App

study of next.js app

# setup

```
$ npm install react react-dom next
$ npm install @types/react-dom @types/node typescript -D
```

# usage

## start dev

$ npx next

## build

$ npx next build

## run production mode

$ npx next start

## export static site

$ npx next build
$ npx next export
